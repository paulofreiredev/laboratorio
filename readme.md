# Laboratório de eletrónica 🕹️ 
## Objetivo
Revisar/Aprender eletrônica e documentar as experiências e laboratórios.
## Estrutura
-  `/arduino` pasta aonde ficaram os projetos criados pela IDE do arduino divido nas subpastas design-circuito, aonde tera foto e pinagem e código aonde ficará o codigo do arduino
 - `/app` pasta aonde ficaram codigos em outras linguagens de programação que farão alguma interação com o arduino
